NavBar

Uma barra de navegação é uma seção de uma interface gráfica do usuário destinada a ajudar os visitantes a acessar informações. As barras de navegação são implementadas nos navegadores de arquivos, na web e como elemento de design de alguns sites.

DropDown

Um menu drop-down ou menu suspenso é um elemento de interface que é similar a uma lista, que permite que o usuário escolha um valor de uma lista de opções que "cai para baixo". Quando o menu drop-down está inativo, ele esconde as opções do menu, economizando espaço na tela.
